# STR COVID-19 Analysis

Este repositorio contiene notebooks y scripts para preparar datos de COVID-19, calcular gráficas de proximidad geográfica y ejecutar simulaciones S/E/I/R a nivel país.

## Datos

Los datos de origen se obtienen desde Kaggle:

- [Novel Corona Virus 2019 Dataset](https://www.kaggle.com/datasets/sudalairajkumar/novel-corona-virus-2019-dataset?resource=download)

Descarga el dataset y colócalo en la carpeta `data/` siguiendo la misma estructura utilizada en este proyecto.
